Hur jag skapat denna webbplats

Så här skapade jag denna sida:

  1. Jag började med att initiera ett NPM-projekt i min arbetsmapp. Detta gjorde jag genom att först navigera till min mapp i CMD sen skriva "npm init"
  2. Använde "npm install parcel --save-dev" för att lägga till Parcel i projektet.
  3. Sedan skapade jag en .gitignore-fil som innehöll
    • node_modules/
    • dist/
    • .parcel-cache/
  4. gjorde ändringar i package.json
    • "browserslist": "> 0.5%, last 2 versions, not dead",
    • "start": "parcel",
    • "build": "parcel build"
    • och ändrade "devDependencies"
  5. Sen skapade jag ett nytt repo på Github. Jag skapade efter ett tag en ny branch som heter "utveckling" där jag laddade upp allt.
  6. Jag skapade en sharp.config.json och satte bildkavliteten till 80%.
  7. Jag skapade en mapp som döptes till src och strukturerade en bra mappstruktur.
  8. Jag använde Visual Studio Code för att skapa hemsidan. Jag använde HTML, CSS och JS.
  9. Jag lade till 5 JPG-bilder med picture-tagen och dessa konverteras till AVIF och WebP.
  10. Jag sammanfogade branch "utveckling" och "master".
  11. Kopplade GitHub-repot till Netlify och publicerade webbplatsen med kommandot npm run build.

Mina ändringar leder till:

  • Minifiering av CSS.
  • Transpilering och minifiering av JavaScript.
  • Minifiering av HTML.
  • Optimering av bilder.
  • Bakåtkompabilitet i genererade filer
  • Kopiering av arbetsfiler till mapp för publicering med optimerade filer
